# 🐼 Study Panda

Full-stack study timer app with React frontend and FastAPI backend.

## Project Structure

- `/frontend` - React + Vite + Material UI
- `/backend` - FastAPI + Python + SQLite

## Run Locally

### Backend
```bash
cd backend
pip install -r requirements.txt
fastapi dev
```
### Frontend
```bash
cd frontend
npm install
npm run dev
